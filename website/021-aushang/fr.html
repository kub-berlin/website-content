Protection des données

Chères utilisatrices, chers utilisateurs des offres de la KuB,

Dans le cadre de l'UE, de nouvelles règles concernant la protection des données sont applicables depuis mai 2018 (Règlement général sur la protection des données - RGPD). Nous devons donc vous communiquer les informations suivantes:

  • Pourquoi avons-nous besoin de ces données?
  • Quand et comment vos données sont-elles sauvegardées?
  • Est-ce que nous allons transmettre vos données à des tiers?

Vous serez aussi informés de vos droits.

Pourquoi avons-nous besoin de ces données?

Pour pouvoir vous donner de bons conseils, nous avons parfois besoin d'informations sur vous (données à caractère personnel). Ça pourrait par exemple être :

  • nom
  • date de naissance
  • adresse
  • pays d'origine
  • des photocopies de votre passeport,
  • des photocopies des documents de l'office pour les étrangers,
  • des photocopies des documents du tribunal ou de la police,
  • des données de vos avocat/es,
  • des données de vos assistant/es sociaux,
  • des données de vos soutiens.

Le Règlement général sur la protection des données (RGPD) autorise à utiliser ces données selon l'article 6, al. 1, lettre b) RGPD, l'article 6, al. 1, lettre a) RGPD et l'article 9, al. 2, lettre a) RGPD.

Où nous envoyons vos données ?

Nous envoyons vos données seulement à d'autres personnes ou instances officielles si vous êtes d'accord. Il peut s'agir d'avocat*es, d'instances officielles ou de tribunaux, de médecins*, d'assistant*es sociaux ou de soutiens.

Pendant de temps sauvegardons-nous vos données ?

Nous sauvegardons vos données pour le temps dont nous en avons besoin pour votre consultation. Les documents numériques sont effacés après 3 ans. Les dossiers sont détruits trois ans après la dernière consultation.

Quels sont vos droits ?

  • Vous avez un droit d'information sur les données que nous conservons (article 15 RGPD).
  • Vous avez le droit de faire corriger les informations erronées (article 16 RGPD).
  • Vous avez le droit de faire supprimer vos données (article 17 RGPD).
  • Vous avez le droit de restreindre le traitement de vos données (article 18 RGPD).
  • Vous avez le droit d'accéder à vos données (article 20 RGPD).
  • Vous avez le droit de révoquer votre consentement (article 7, al. 3 RGPD).

Si vous estimez que nous ne respectons pas la loi dans le traitement de vos données à caractère personnel, vous pouvez adresser une réclamation à l'autorité pour la protection des données :
